# Push to GitHub Instructions

## Quick Push

```bash
# 1. Navigate to project directory
cd UKLtdRegistration-GitHub

# 2. Initialize Git
git init
git branch -M main

# 3. Configure Git
git config user.name "Your Name"
git config user.email "your.email@example.com"

# 4. Add all files
git add .

# 5. Commit
git commit -m "Complete UK LTD Registration Platform v2.0 - Updated 2026 fees, 20+ services, affiliate system, legal compliance"

# 6. Add remote
git remote add origin https://github.com/FazalShahidLatif/UKLtdRegistration.git

# 7. Push to GitHub
git push -u origin main
```

## Alternative: GitHub Desktop

1. Download GitHub Desktop
2. File → Add Local Repository
3. Select `UKLtdRegistration-GitHub` folder
4. Create repository on GitHub.com first
5. Publish repository

## Alternative: Upload via Web

1. Visit https://github.com/FazalShahidLatif/UKLtdRegistration
2. Click "uploading an existing file"
3. Drag and drop all files
4. Commit changes
