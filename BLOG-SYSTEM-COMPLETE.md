### Schema Markup (All Articles) ✅

**1. Article Schema:**
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "...",
  "datePublished": "2026-03-12",
  "dateModified": "2026-03-12",
  "author": {...},
  "publisher": {...}
}
```

**2. FAQPage Schema** (for articles with FAQs):
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [...]
}
```

**3. HowTo Schema** (for guides):
```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "...",
  "step": [...]
}
```

**4. BreadcrumbList Schema:**
```json
{
  "@context": "https://schema.org",

  "itemListElement": [...]
}
```
